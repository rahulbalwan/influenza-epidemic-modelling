## Conceptual Summary

This project integrates:

- statistical modelling (growth rate estimation)  
- epidemiological theory (R₀, compartmental models)  
- mechanistic modelling (SEIR, SIRS)  
- seasonal dynamics (β(t))  

to understand influenza as a **recurrent seasonal epidemic system**.
